```C++
class Solution {
public:
    vector<int> twoSum(vector<int>& numbers, int target) {
        int *left = numbers.data();//numbers的头元素
        int *right = (numbers.data()+numbers.size())-1;//numbers的末尾元素
        while(left<right){
            int s = *left + *right;
            if(s == target){
            int leftIndex=(left - numbers.data()+1);//题目要的是从1开始的位置 所以＋1
            int rightIndex=(right - numbers.data()+1);
            return {leftIndex,rightIndex};
            }
            if(s>target){
                right -=1;
            }else{
                left+=1;
            }
        }
        return{};
    }
};
```

# LeetCode 15. 三数之和：排序 + 双指针笔记

## 题目要求

给定整数数组 `nums`，找出所有不重复的三元组：

```cpp
nums[i] + nums[j] + nums[k] == 0
```

并且要求：

```cpp
i != j
i != k
j != k
```

返回值类型是：

```cpp
vector<vector<int>>
```

因为答案是多个三元组，例如：

```cpp
[[-1, -1, 2], [-1, 0, 1]]
```

---

## 一、最开始的错误：`sort()` 不能赋值给 `ans`

错误写法：

```cpp
vector<int> ans;
ans = sort(nums.begin(), nums.end());
```

### 错误原因

`sort(nums.begin(), nums.end())` 是**原地排序**。

也就是说，它会直接修改 `nums` 本身，不会返回一个新的数组。

所以不能写：

```cpp
ans = sort(...);
```

正确写法：

```cpp
sort(nums.begin(), nums.end());
```

---

## 二、`ans` 的类型应该是 `vector<vector<int>>`

错误写法：

```cpp
vector<int> ans;
```

因为题目返回的是多个三元组，所以应该是二维数组：

```cpp
vector<vector<int>> ans;
```

每找到一个三元组，就放进去：

```cpp
ans.push_back({nums[i], nums[left], nums[right]});
```

---

## 三、`sort(nums.begin(), nums.end())` 默认是从小到大排序

例如：

```cpp
nums = {-1, 0, 1, 2, -1, -4};
```

排序后变成：

```cpp
nums = {-4, -1, -1, 0, 1, 2};
```

注意：默认不是从大到小，是**从小到大**。

如果想从大到小，要写：

```cpp
sort(nums.begin(), nums.end(), greater<int>());
```

但是三数之和通常使用从小到大排序。

---

## 四、核心思路：排序 + 双指针

固定一个数 `nums[i]`，然后在它右边找两个数：

```cpp
nums[i] + nums[left] + nums[right] == 0
```

排序后：

```cpp
i < left < right
```

例如：

```cpp
[-4, -1, -1, 0, 1, 2]
```

固定：

```cpp
nums[i] = -1
```

然后让：

```cpp
left = i + 1;
right = n - 1;
```

不断计算：

```cpp
sum = nums[i] + nums[left] + nums[right];
```

### 判断逻辑

如果：

```cpp
sum == 0
```

说明找到一个答案。

如果：

```cpp
sum < 0
```

说明总和太小，需要让它变大。

因为数组已经从小到大排序，所以移动左指针：

```cpp
left++;
```

如果：

```cpp
sum > 0
```

说明总和太大，需要让它变小。

所以移动右指针：

```cpp
right--;
```

---

## 五、下标版正确代码

建议先掌握这个版本。

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> ans;
        sort(nums.begin(), nums.end());
        int n = nums.size();
        for (int i = 0; i < n; i++) {
            // 如果 nums[i] 已经大于 0
            // 后面的数也都 >= nums[i]
            // 三个正数不可能加出 0
            if (nums[i] > 0) {
                break;
            }
            // i 去重
            // 避免固定相同的 nums[i]
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            int left = i + 1;
            int right = n - 1;
            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                if (sum == 0) {
                    ans.push_back({nums[i], nums[left], nums[right]});
                    // left 去重
                    while (left < right && nums[left] == nums[left + 1]) {
                        left++;
                    }
                    // right 去重
                    while (left < right && nums[right] == nums[right - 1]) {
                        right--;
                    }
                    // 找到答案后，左右指针都要移动
                    left++;
                    right--;
                }
                else if (sum < 0) {
                    left++;
                }
                else {
                    right--;
                }
            }
        }
        return ans;
    }
};
```

## 六、为什么要去重？

题目要求：

```cpp
答案中不可以包含重复的三元组
```

例如：

```cpp
[-1, -1, 0, 1, 2]
```

可能会重复找到：

```cpp
[-1, 0, 1]
```

所以要去重。

---

## 七、三个去重位置

### 1. 固定数 `i` 去重

```cpp
if (i > 0 && nums[i] == nums[i - 1]) {
    continue;
}
```

意思是：

如果当前固定的数和上一个一样，就跳过。

例如：

```cpp
[-4, -1, -1, 0, 1, 2]
```

第一个 `-1` 已经处理过了，第二个 `-1` 就不用再处理。

---

### 2. `left` 去重

找到答案后：

```cpp
while (left < right && nums[left] == nums[left + 1]) {
    left++;
}
```

意思是：

如果左指针后面还有相同的数，就跳过去，避免重复答案。

---

### 3. `right` 去重

找到答案后：

```cpp
while (left < right && nums[right] == nums[right - 1]) {
    right--;
}
```

意思是：

如果右指针前面还有相同的数，也跳过去。

---

## 八、为什么 `sum < 0` 要 `left++`？

数组已经排序：

```cpp
[-4, -1, -1, 0, 1, 2]
```

如果：

```cpp
sum < 0
```

说明当前三个数加起来太小。

想让总和变大，只能移动左指针。

因为 `left++` 会让左边的数变大。

例如：

```cpp
left 指向 -1
left++ 后可能指向 0
```

所以：

```cpp
sum < 0 -> left++;
```

---

## 九、为什么 `sum > 0` 要 `right--`？

如果：

```cpp
sum > 0
```

说明当前三个数加起来太大。

想让总和变小，就移动右指针。

因为 `right--` 会让右边的数变小。

例如：

```cpp
right 指向 2
right-- 后可能指向 1
```

所以：

```cpp
sum > 0 -> right--;
```

## 十、指针版容易出错的地方

你写过类似代码：

```cpp
int *right = nums.data() + nums.size() - 1;
int *left = nums.data() + 1;
```

这时候：

```cpp
left
right
```

它们是指针，不是下标。

## 十一、为什么 `nums[left]` 会报错？

错误写法：

```cpp
int sum = x + nums[left] + nums[right];
```

报错原因：

```cpp
nums[left]
```

要求 `left` 是下标，例如：

```cpp
nums[0]
nums[1]
nums[2]
```

但是你这里的 `left` 是指针：

```cpp
int* left
```

所以不能作为 `vector` 的下标。

---

## 十二、指针应该用 `*left` 取值

如果 `left` 是指针，那么取它指向的值要写：

```cpp
*left
```

所以错误写法：

```cpp
nums[left]
nums[right]
```

应该改成：

```cpp
*left
*right
```

例如：

```cpp
int sum = *i + *left + *right;
```

---

## 十三、`*i` 是值，不是下标

错误写法：

```cpp
nums[*i]
```

原因：

```cpp
*i
```

是当前指针指向的元素值。

例如排序后：

```cpp
nums = {-4, -1, -1, 0, 1, 2}
```

如果 `i` 指向第一个元素：

```cpp
*i == -4
```

那么：

```cpp
nums[*i]
```

就变成：

```cpp
nums[-4]
```

这是错误的，因为数组下标不能是负数。

## 十四、指针版正确写法

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> ans;

        sort(nums.begin(), nums.end());

        int* start = nums.data();
        int* end = nums.data() + nums.size();

        for (int* i = start; i < end; i++) {
            if (*i > 0) {
                break;
            }

            // i 去重
            if (i > start && *i == *(i - 1)) {
                continue;
            }

            int* left = i + 1;
            int* right = end - 1;

            while (left < right) {
                int sum = *i + *left + *right;

                if (sum == 0) {
                    ans.push_back({*i, *left, *right});

                    while (left < right && *left == *(left + 1)) {
                        left++;
                    }

                    while (left < right && *right == *(right - 1)) {
                        right--;
                    }

                    left++;
                    right--;
                }
                else if (sum < 0) {
                    left++;
                }
                else {
                    right--;
                }
            }
        }

        return ans;
    }
};
```

## 十五、指针版和下标版的区别

### 下标版

```cpp
int left = i + 1;
int right = n - 1;

nums[left]
nums[right]
```

这里 `left` 和 `right` 是下标。

### 指针版

```cpp
int* left = i + 1;
int* right = end - 1;

*left
*right
```

这里 `left` 和 `right` 是指针。

这里的 `left` 必须是整数下标。

如果 `left` 是指针，就要写：

## 十七、你当前代码里的几个问题总结

### 1. 指针和下标混用了

你定义的是：

```cpp
int* left;
int* right;
```

但是后面用了：

```cpp
nums[left]
nums[right]
```

这是错误的。

---

### 2. `x` 的作用域有问题

你写过类似：

```cpp
if (...) {
    int x = nums[*i];
    continue;
}

int sum = x + nums[left] + nums[right];
```

问题是：

```cpp
int x
```

定义在 `{}` 里面，只能在这个大括号里面使用。

出了大括号，`x` 就不存在了。

所以不能在外面继续用 `x`。

更推荐直接写：

```cpp
int sum = *i + *left + *right;
```

或者下标版：

```cpp
int sum = nums[i] + nums[left] + nums[right];
```

---

### 3. `right--` 和 `left++` 的位置不对

你一开始在 `for` 循环里面写：

```cpp
right--;
left++;
```

这样会导致每轮循环还没开始找答案，指针就乱动了。

正确做法是：

每次固定一个 `i` 后，重新设置：

```cpp
left = i + 1;
right = n - 1;
```

也就是：

```cpp
for (int i = 0; i < n; i++) {
    int left = i + 1;
    int right = n - 1;

    while (left < right) {
        ...
    }
}
```

---

### 4. `ans.push_back()` 不能为空

错误写法：

```cpp
ans.push_back();
```

正确写法：

```cpp
ans.push_back({nums[i], nums[left], nums[right]});
```

或者指针版：

```cpp
ans.push_back({*i, *left, *right});
```

## 十九、核心口诀

三数之和：

```text
先排序，固定一个数，剩下两个数用双指针找。
sum 小了 left++，sum 大了 right--。
找到答案后，left 和 right 都要去重。
```

指针和下标：

```text
下标版：nums[left]
指针版：*left
不能混着用。
```

`sort()`：

```text
sort 是原地排序，没有返回值。
不能 ans = sort(...)
只能 sort(nums.begin(), nums.end())
```

返回值：

```text
一个三元组：vector<int>
多个三元组：vector<vector<int>>
```
