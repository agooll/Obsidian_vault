### 题解
题目要求最多出现2次的最长字符串 与滑动窗口使用哈希map的原理一样 可以定义一个哈希map 然后去遍历数组的元素 记录 mp[s[right]]>2的元素 然后找出最长字串

#### LeetCode解法
```cpp
class Solution {
public:

    int maximumLengthSubstring(string s) {

        unordered_map<char,int>mp;

        int n = s.size();

        int left = 0;

        int ans = 0;

        for(int right = 0;right<n;right++){

            mp[s[right]]++;

            while(mp[s[right]]>2){

                mp[s[left]]--;

                left++;

            }

            ans = max (ans,right-left+1);

        }

        return ans;

    }

};
```
