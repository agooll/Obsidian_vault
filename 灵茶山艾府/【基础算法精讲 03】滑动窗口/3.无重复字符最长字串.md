### unordered_set 写法
```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_set<char>window;
        int ans = 0;
        int left = 0;
        int n = s.size();
        for(int right = 0; right < n;right++){
            while(window.count(s[right])){
                window.erase(s[left]);
                left++;
            }
            window.insert(s[right]);
            ans = max(ans,right - left + 1);
        }
        return ans;
    }
};
```
#### while(window.count(s[right])){}为什么可以作为一个判断条件
count是哈希set的一个API 由于哈希set的特性 他不能保存相同值 所以count的返回值只会是 0 或
1 所以这个的判断条件是window.count(s[right] 会不会等于 1 等于1 为true 即存在这个元素就为true
#### window.erase()的理解
这个是删除元素的API 
### unordered_map写法
```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char, int> mp;
        int left = 0;
        int ans = 0;
        for (int right = 0; right < s.size(); right++) {
            mp[s[right]]++;
            while (mp[s[right]] > 1) {
                mp[s[left]]--;
                left++;
            }
            ans = max(ans, right - left + 1);
        }
        return ans;
    }
};
```
#### LeetCode 3. 无重复字符的最长子串：哈希 map 写法

##### 题目核心

给定一个字符串 `s`，要求找出其中 **不含重复字符的最长子串长度**。

注意：

- 子串：必须是连续的
- 子序列：可以不连续

例如：

```text
s = "pwwkew"
```

最长无重复子串是：

```text
"wke"
```

长度为 `3`。

而 `"pwke"` 虽然没有重复字符，但它不是连续的，所以不是子串。

---

##### 核心思路：滑动窗口 + 哈希 map

这道题可以用滑动窗口解决。

窗口用两个指针表示：

```cpp
left
right
```

窗口范围是：

```cpp
[left, right]
```

也就是当前正在检查的一段子串。

---

##### 哈希 map 的作用

使用：

```cpp
unordered_map<char, int> mp;
```

它的作用是记录：

```text
当前窗口中，每个字符出现了几次
```

例如当前窗口是：

```text
abca
```

那么 map 里面大概是：

```text
a : 2
b : 1
c : 1
```

因为 `a` 出现了两次，所以当前窗口不合法。

---

##### 为什么要用 map？

因为我们要判断：

```text
当前窗口中有没有重复字符
```

如果某个字符出现次数大于 1，说明重复了。

判断条件就是：

```cpp
mp[s[right]] > 1
```

含义是：

```text
当前 right 指向的字符，在窗口中出现次数超过 1 次
```

说明它重复了。

---

##### 整体逻辑

###### 1. right 负责扩大窗口

每次让 `right` 往右走，把新字符加入窗口：

```cpp
mp[s[right]]++;
```

意思是：

```text
当前字符出现次数 +1
```

---

###### 2. 如果出现重复，left 负责缩小窗口

如果：

```cpp
mp[s[right]] > 1
```

说明当前字符重复了。

这时不断移动 `left`，把左边的字符移出窗口：

```cpp
mp[s[left]]--;
left++;
```

直到当前字符不重复为止。

---

###### 3. 每次窗口合法后，更新答案

窗口长度是：

```cpp
right - left + 1
```

所以更新最大长度：

```cpp
ans = max(ans, right - left + 1);
```

---

##### 完整代码：哈希 map 写法

```cpp
#include <iostream>
#include <string>
#include <unordered_map>
#include <algorithm>
using namespace std;

class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char, int> mp;

        int left = 0;
        int ans = 0;

        for (int right = 0; right < s.size(); right++) {
            // 1. 把 right 指向的字符加入窗口
            mp[s[right]]++;

            // 2. 如果当前字符重复了，就移动 left 缩小窗口
            while (mp[s[right]] > 1) {
                mp[s[left]]--;
                left++;
            }

            // 3. 当前窗口已经没有重复字符，更新最大长度
            ans = max(ans, right - left + 1);
        }

        return ans;
    }
};
```

---

##### 代码逐句解释

###### 定义哈希 map

```cpp
unordered_map<char, int> mp;
```

含义：

```text
key：字符
value：这个字符在当前窗口中出现的次数
```

例如：

```text
mp['a'] = 2
```

表示：

```text
字符 'a' 在当前窗口中出现了 2 次
```

---

###### 定义左指针和答案

```cpp
int left = 0;
int ans = 0;
```

含义：

```text
left：窗口左边界
ans：目前找到的最长无重复子串长度
```

---

###### 右指针遍历字符串

```cpp
for (int right = 0; right < s.size(); right++)
```

含义：

```text
right 从左到右扫描字符串
每次把 s[right] 加入窗口
```

---

###### 把当前字符加入窗口

```cpp
mp[s[right]]++;
```

含义：

```text
当前字符的出现次数 +1
```

例如：

```text
s[right] = 'a'
```

那么：

```cpp
mp['a']++;
```

---

###### 判断是否重复

```cpp
while (mp[s[right]] > 1)
```

含义：

```text
只要当前 right 指向的字符出现次数大于 1，
就说明当前窗口中有重复字符。
```

为什么只看 `s[right]`？

因为在加入 `s[right]` 之前，窗口本来是合法的，没有重复字符。

现在出现重复，一定是因为新加入的 `s[right]` 导致的。

所以只需要判断：

```cpp
mp[s[right]] > 1
```

---

###### 移动左边界

```cpp
mp[s[left]]--;
left++;
```

含义：

```text
把窗口最左边的字符移出去
然后 left 往右移动一格
```

例如窗口是：

```text
a b c a
^     ^
left  right
```

此时 `a` 重复了。

执行：

```cpp
mp[s[left]]--;
left++;
```

就是把最左边的 `a` 移出窗口。

窗口变成：

```text
b c a
```

此时窗口重新合法。

---

##### 示例演示：s = "abcabcbb"

初始：

```text
left = 0
ans = 0
mp 为空
```

---

###### right = 0，字符是 'a'

```text
窗口：a
mp: a = 1
ans = 1
```

---

###### right = 1，字符是 'b'

```text
窗口：ab
mp: a = 1, b = 1
ans = 2
```

---

###### right = 2，字符是 'c'

```text
窗口：abc
mp: a = 1, b = 1, c = 1
ans = 3
```

---

###### right = 3，字符是 'a'

加入新的 `a` 后：

```text
窗口：abca
mp: a = 2, b = 1, c = 1
```

此时 `a` 重复了，所以移动 `left`。

删除左边的 `a`：

```text
窗口：bca
mp: a = 1, b = 1, c = 1
```

窗口重新合法。

```text
ans = 3
```

---

##### 为什么 while 里面是 `mp[s[right]] > 1`

```cpp
while (mp[s[right]] > 1)
```

意思是：

```text
只要当前字符重复，就一直删左边的字符
直到当前字符不重复为止
```

不能只用 `if`，因为有些时候要删多个字符。

例如：

```text
s = "abba"
```

当 right 指向最后一个 `a` 时，可能需要移动多次 left，直到窗口重新合法。

所以这里用 `while` 更稳。

---

##### 这题的本质

这题不是单纯地找有没有重复字符，而是要维护一个窗口：

```text
窗口里面永远不能有重复字符
```

所以：

```text
right 负责扩张窗口
left 负责缩小窗口
map 负责记录字符出现次数
ans 负责记录最大窗口长度
```

---

##### 记忆口诀

```text
右指针进窗口，次数加一；
如果字符重复，左指针出窗口；
窗口合法之后，更新最大长度。
```

---

##### 和 unordered_set 写法的区别

###### unordered_set 写法

只记录：

```text
字符在不在窗口里
```

例如：

```cpp
unordered_set<char> window;
```

判断方式：

```cpp
window.count(s[right])
```

返回值：

```text
0：不存在
1：存在
```

---

###### unordered_map 写法

记录：

```text
字符在窗口里出现了几次
```

例如：

```cpp
unordered_map<char, int> mp;
```

判断方式：

```cpp
mp[s[right]] > 1
```

含义：

```text
当前字符出现次数超过 1，说明重复了
```

---

##### 推荐初学者理解方式

初学时更推荐先理解 `unordered_map<char, int>` 写法。

因为它更直观：

```text
字符出现几次，就记录几次。
超过 1，就说明重复。
```

等 map 写法熟了，再去理解 set 写法会更容易。
