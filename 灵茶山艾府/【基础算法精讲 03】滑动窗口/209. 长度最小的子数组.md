#### LeetCode代码

```cpp
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int n = nums.size();
        int sum = 0,left =0,ans = n+1;
        //i是向右移动的右端点
        for(int i= 0;i<n;i++){
            sum += nums[i];
            while(sum-nums[left]>=target){
                sum -=nums[left];
                left++;
            }
            if(sum>=target){
                ans = min(ans,i-left+1);
            }
        }
        return ans<=n?ans:0;
    }
};
```
##### ans为什么要初始化为n+1;
因为题目要求的是求最小值 如果定义为0的话 到最后min(ans,i-left+1)的时候 他就只能一直是0 题目也讲了是正整数 所以定义为0是错的;
##### min(ans,i-left+1)；为什么这里要加1?
这里的窗口是 闭区间 [left, i]，左右两个端点都算进子数组里，所以长度要写：
i - left + 1
为什么不是 i - left
i - left 算的是“两个下标之间差了几个位置”，不是“元素个数”。
比如：
left = 2, i = 4
对应子数组是：
nums[2], nums[3], nums[4]
一共有 3 个元素。
但：
i - left = 4 - 2 = 2
少算了一个。
所以要：
4 - 2 + 1 = 3
再举个最小例子
==如果：==
==left = 3, i = 3==
==说明窗口里只有一个元素 nums[3]。==
==这时候长度应该是 1，而不是 0：==
==i - left + 1 = 3 - 3 + 1 = 1==
一句话记忆
闭区间 [l, r] 长度：r - l + 1
半开区间 [l, r) 长度：r - l
你这道题里窗口显然是：
[left, i]
所以必须 +1。
