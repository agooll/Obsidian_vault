#### LeetCode代码
```cpp
class Solution {
public:
    int numSubarrayProductLessThanK(vector<int>& nums, int k) {
        int n = nums.size();
        int ans = 0;
        int left = 0;
        if(k<=1){
            return 0;
        }
        int x = 1;
        for(int right = 0;right<n;right++){
            x *= nums[right];
            while(x>=k){
                x/=nums[left];
                left++;
            }
            ans += right -left +1;
        }
        return ans;
    }
};
```
##### 为什么是/=？
因为这道题是求窗口积的 元素要出去 必须==/= ==
如果是窗口和 元素出去==**-=**==
##### 为什么x不能在for里面定义
1. x 放进 for 里会每轮重置
滑动窗口里，x 表示当前窗口 [left, right] 的乘积。
也就是：
x = nums[left] * nums[left+1] * ... * nums[right]
所以当 right 往右走时，新的 x 应该在旧的基础上更新：
x *= nums[right];
如果你写成：
for (...) {
    int x = 1;
    ...
}
那每次循环 x 都重新变成 1，前一轮窗口的信息就没了。
