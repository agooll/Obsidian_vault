# 4. 文件系统与磁盘调度

- 第 8 题：站在用户观点来看，大家不想去记磁盘的物理块号，操作系统引入文件系统就是为了让用户能实现按名存取。

- 第 9、15 题：磁盘访问时间由寻道时间、旋转延迟时间和传输时间组成。
  - 指定扇区转到磁头下的时间是旋转时间。
  - 磁盘调度算法（如 SCAN、CSCAN、SSTF）的核心目的全是为了减少磁头移动的距离，即缩短寻道时间。

- 第 13 题：SCAN（电梯调度）算法。当前在 105 道且向增加方向移动，所以磁头会像电梯一样先一路向上把大的请求做完（110 -> 170 -> 180 -> 195），到头后再掉头一路向下把小的请求做完（68 -> 45 -> 35 -> 12）。
