## 刷题路径

### 总原则

- 不建议直接按 Hot100 原顺序刷。
- 也不建议单纯按“简单 -> 中等 -> 困难”刷。
- 最适合你的方式：**按专题刷，用 Hot100 当题库；每个专题内部再按简单 -> 中等 -> 偏难刷。**

### 你现在最适合的顺序

1. 数组 / 哈希
2. 双指针
3. 滑动窗口
4. 前缀和
5. 栈 / 单调栈
6. 链表
7. 二叉树
8. 二分查找
9. 回溯
10. 堆 / 优先队列

### 第一轮建议刷法

- 每个专题先做 2~3 道简单题，把模板做熟。
- 再做 5~8 道中等题，练成“看到题能想到解法”。
- 暂时不要死磕难题，优先把中等题刷顺。
- 每做完一个专题，回头复盘一次错题和模板。

### 使用建议

- 每天 2 题，优先保证“做完 + 复盘”，不要只追数量。
- 做完一道题后，给自己标一个结果：
  - 会做
  - 看题解后会做
  - 还不会
- 如果连续 3 道都得看题解，说明这个专题还不稳，先别急着跳下一个专题。
- 每刷完 10 题，回头复盘一遍错题和模板。
