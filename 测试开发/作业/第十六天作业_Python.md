#### 1.python的函数怎么传入可变参数 或 传参时的 `*`和 `**`是什么意思?
`*` 和 `**` 用于处理函数的可变参数，主要分为“定义函数”和“调用函数”两种场景：
##### 1.在函数定义时 用于接收不定数量的参数
`*` （单星号）会将多个位置参数打包成元组
例如：
```
def func(*agrs):print(args)
调用func(1,2,3) 输出(1,2,3)
```
** （双星号）接受多余的关键字参数
会将形如`key = value`的参数打包成一个字典
例如：
```
def func(**kwargs): print(kwargs)
调用func(a=1,b=2) 输出{'a' = 1,'b'=2}
```
##### 2.在函数调用时 用于解包数据结构的操作
`*`  解包列表或者元组
它会将列表或者元组中的每个元素按顺序传给函数的位置参数
`**` 解包字典
它会将字典的键值对按名称传给对应的关键字参数。
#### 2.局部变量和全局变量的区别
1.定义域
全局变量的定义是在所有函数的外部
局部变量的定义是在函数的内部
2.作用域
全局变量的作用域是整个*.py文件内都可以使用
局部变量的作用域是在函数的内部使用
3.不同函数之间内能使用的变量只有各自函数内定义的局部变量跟全局变量
想要跨函数调用变量 那变量必须得是全局变量
4.如果局部变量和全局变量同名 函数内部会优先使用局部变量
#### 3.面向对象和面向过程的区别？
**面向过程**
	**“怎么做”** 它将问题拆解成一个个具体的步骤 然后通过**函数**一步步调用 
**面向对象**
	**特点**：具有**封装、继承、多态**三大特性  强调的是 **“谁来做”** 它将问题看作是由多个**对象**协作完成的 每个对象都有自己的属性和行为
#### 4.类和对象的关系
1.类是对象的模板 对象是类创建出来的具体实例 
2.一个类可以有多个对象
3.类用来描述一类事物共同拥有的属性和行为
4.对象是根据类创建出来的具体个体
