#### 1.python的函数怎么传入可变参数 或 传参时的 `*`和 `**`是什么意思?
`*` 和 `**` 用于处理函数的可变参数，主要分为“定义函数”和“调用函数”两种场景：
1.在函数定义时 用于接收不定数量的参数
`*` （单星号）会将多个位置参数打包成元组
例如：
```
def func(*agrs):print(args)
调用func(1,2,3) 输出(1,2,3)
```
