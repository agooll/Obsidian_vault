# 笔记整理快速指令

## 通用版

```text
按“Obsidian笔记整理Skill”处理 `xxx.md`：
- 去头去尾噪音，中间正文不删
- 修复 Markdown 显示（列表、空行、编号）
- 按内容加多级标题，空标题不要
- 输出时给 文件路径 + 行号
```

## 紧凑版（推荐）

```text
按“Obsidian笔记整理Skill-紧凑版”处理 `xxx.md`：
- 保留正文知识点，不改语义
- 列表项之间不留空行
- 标题前后仅保留必要空行
- 清理零宽字符
- 输出关键改动行号
```

## JMeter/F12 映射补写版

```text
按“Obsidian笔记整理Skill”处理 `xxx.md`，并补写：
- F12抓包字段 -> JMeter对应区域对照表
- Authorization 两套模板（token自带Bearer/不自带Bearer）
- 常见错误排查清单
- 输出写入章节和行号
```
